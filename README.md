## ✨ Features

- 🎵 Play, pause, skip track controls
- 🔊 Volume slider and mute toggle
- 📋 Playlist sidebar with song list
- ⏱️ Progress bar with seek functionality
- 🎨 Spotify-accurate dark UI design

## 🚀 Run Locally

```bash
git clone https://github.com/aasimansari1/Spotify-Clone.git
# Open index.html in your browser
```
